# AI Utility Hub

**AI Utility Hub** is a full-stack **AI SaaS app** built with **MERN** (MongoDB, Express, React, Node) offering multiple AI tools and subscription-based payments.

## Features
- **Auth & Users:** Signup/login with JWT, bcrypt, protected routes.  
- **AI Tools:** Notes Summarizer, Code Debugger, Interview Simulator, Resume Analyzer, Caption/Content Generator.  
- **Dashboard:** Sidebar for services, main area for AI results, history panel, responsive design.  
- **AI Integration:** OpenAI / Gemini API, responses saved in MongoDB.

## Tech Stack
- **Frontend:** React.js, Tailwind/Material-UI  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB
